Minishell

A simplified Unix shell written in C as part of the 42 school curriculum. This project recreates the behavior of a basic command-line interpreter, including command execution, environment variable handling, pipes, and input/output redirections. It also focuses on process management, parsing, and low-level system calls to better understand how a real shell works internally.

Cub3d

A 3D raycasting project written in C, inspired by classic games like Wolfenstein 3D. The goal is to render a first-person view inside a 2D map by simulating a 3D environment using the DDA (Digital Differential Analyzer) algorithm for efficient raycasting. The project includes wall rendering, player movement, collision detection, and texture mapping, providing a deeper understanding of graphics rendering, linear algebra, and low-level game engine mechanics.

Philosophers

A concurrency simulation written in C based on the classic dining philosophers problem. The project focuses on multithreading (or multiprocessing), synchronization, and resource sharing using mutexes. It simulates philosophers who alternate between thinking, eating, and sleeping while avoiding deadlocks and starvation. The main goal is to understand thread management, race conditions, and efficient synchronization in low-level systems programming
